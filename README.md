# BIOTERRA WEB

Sitio web base para Bioterra México como consultora ambiental.

## Incluye
- index.html
- assets/css/styles.css
- assets/js/script.js
- assets/js/courses-data.js
- assets/images/brand
- assets/images/courses con nombres optimizados
- assets/images/gallery con nombres optimizados
- docs/contactos-originales.txt

## Secciones
- Inicio
- Cursos
- ¿Quiénes somos?
- Galería
- Contacto

## Funciones integradas
- Animaciones al hacer scroll
- Tarjetas interactivas
- Modal para cursos
- Lightbox de galería
- Botones flotantes de WhatsApp, Facebook, Instagram y Gmail
- Diseño responsive para celular
